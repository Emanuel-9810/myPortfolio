Project: Blog Application (Android)

Health App

This android application was worked on with a group of peers for a database design project. The application utilizes Java and Sqlite as the database language. The application was created for users to create blogs, comment in blogs, and keep track of interactions between users such as taking positive/negative comments, display certain user activity(user's followers, users with positive comments, users who never posted...), and others.

My role in this project was to create the login page, the commenting features, and help with the design. This project was primarily worked on with members collaborating to finish features quickly and more effecient.
